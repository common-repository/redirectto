Plugin to redirect pages or posts to another sites, pages or posts after some seconds that you can delegate.

== Description ==

This is a plugin used to redirect people for another page, post or site after some seconds .

You can set:
* time to be redirect
* site to be redirect
* message explaining why you are redirecting

== Installation ==

Upload the RedirectTo Plugin to your blog and activate it.

To use, just insert the shortcode [redirectTo url="http://yoururl.com" in="30"]And if you want, a message here[/redirectTo]
